Getting MultiLife

Our source code is stored via Git repository on Github.

http://github.com/MurphysChaos/MultiLife/tree/master

If you are using a Unix-like environment and have Git installed, and meet the build requirements (described below), you can get and run MultiLife with the following set of instructions.

    git clone git://github.com/MurphysChaos/MultiLife.git
    qmake
    make
    ./MultiLife

Build Requirements

MultiLife is built using GNU C++ 4.3.3 and Qt 3.3.

Editing Requirements

While you should be able to modify MultiLife with any text editor, if you want to do any work on the main UI, we highly recommend using Qt Designer 3.3 or higher.

Playing the MultiLife

This is a "zero-player" game. Meaning, while the simulation is running, you don't do anything.
